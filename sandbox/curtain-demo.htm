windows

a handy, loosely-coupled jQuery plugin for scrolling windows

what's 'windows'?

This plugin allows you to create a single-page website with multiple 'windows' that unroll much like a curtain rises. To navigate, you can use your keyboard, the scrollbar, or your mousewheel. For a demonstration of the different effects, scroll into this document.
